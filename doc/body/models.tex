\section{有限元模型}
设计有限元的类，需要保存什么？还需要考虑的是怎样保存的问题？进一步地就需要实现怎样去获取这些东西？用户应该如何输入操作来生成这样的数据。
\begin{enumerate}
	\item 求解模型的纬度，坐标系
	\item 单元数，节点数，边界条件数
	\item 单元列表
	\item 节点列表
	\item 初始条件
	\item 边界条件
	\item 材料列表
	\item 求解域内的方程
	\item 求解区域的列表，每一个求解区域应当包含材料，初始条件，边界条件和方程等
	\item 求解器接口
\end{enumerate}
